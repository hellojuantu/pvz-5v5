# PvZ 5v5 Multiplayer 🌻🧟

植物大战僵尸多人对战版 - 支持 5v5 实时对战!

## 特性

- 🎮 **实时多人对战**: 5v5 植物 vs 僵尸
- 🌊 **波次系统**: 自动波次 + 手动购买波次
- 🤖 **智能 AI**: 自动补位机器人
- 🔄 **断线续玩**: 支持断线重连
- ⏸️ **暂停系统**: 游戏内暂停功能
- 📊 **排行榜**: 游戏历史记录

## 安装

```bash

```

访问 http://localhost:3000/multiplayer.html
